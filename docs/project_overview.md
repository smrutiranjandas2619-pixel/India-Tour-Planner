# Project Overview: India Tour Planner AI

An enterprise RAG-based travel routing advisor and smart calculator designed for travelers across India.

## Key Features

1. **RAG Travel Consultant**: Dense retrieval similarity indexing matching 15 Indian travel hubs seamlessly.
2. **Interactive Map Visualizer**: Nominatim geocoding falls back to automatic scenic checkpoints and attraction clustering.
3. **Smart Expense Tracker**: Mathematical room sharing, vehicle rents, fuel estimates, and emergency contingency buffers.
4. **Follow-Up Speech Chatbot**: Immersive voice-enabled RAG travel engine supporting English and 5 regional languages.
5. **Group bill ledger**: settle balances chips dynamically.
